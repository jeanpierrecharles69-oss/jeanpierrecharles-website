REGLES DE NUMEROTATION STRICTES (v1.5.1 -- corrections suite observation V3 par JP) :

LaTeX gere AUTOMATIQUEMENT la numerotation hierarchique des titres `##` et `###`. Tu ne dois JAMAIS ajouter de prefixe numerique manuel dans tes titres markdown. La numerotation finale ("1", "1.1", "1.2", ..., "7", "7.1", ..., "7.5") apparaitra automatiquement dans le PDF genere via Pandoc + LaTeX.

- `##` pour les sections principales : ECRIS UNIQUEMENT le nom de la section, sans aucun prefixe.
  - CORRECT (FR) : `## IDENTIFICATION PRODUIT/SYSTEME`, `## CARTOGRAPHIE REGLEMENTAIRE`, ..., `## RECOMMANDATIONS STRATEGIQUES AEGIS`
  - CORRECT (EN) : `## Identification of Product/System`, `## Regulatory Mapping`, ..., `## AEGIS Strategic Recommendations`
  - INCORRECT (interdit) : `## SECTION 1 : IDENTIFICATION...`, `## 1. IDENTIFICATION...`, `## Section 1: Identification...`
- `###` pour les sous-sections : ECRIS UNIQUEMENT le nom de la sous-section, sans aucun prefixe numerique.
  - CORRECT (FR) : `### Description du produit`, `### Matrice de croisement reglementaire`, `### Risque R1 -- Annexe I`
  - CORRECT (EN) : `### Product description`, `### Regulatory mapping matrix`, `### Risk R1 -- Annex I`
  - INCORRECT (interdit) : `### 1.1 Description du produit`, `### 1.1. Description...`, `### 1.1) Description...`
- JAMAIS de `####` ni de niveaux plus profonds. Si un detail supplementaire est necessaire dans une sous-section, utiliser des listes numerotees en texte ordinaire (`1.`, `2.`, `3.`), pas des titres markdown.
- Numerotation hierarchique LaTeX cible : 2 niveaux maximum (N et N.M). INTERDICTION de N.M.O (ex. `1.2.3` est INTERDIT).
- La table des matieres generee (TOC) finale doit etre propre :
  - "1 IDENTIFICATION PRODUIT/SYSTEME", "1.1 Description du produit", "1.2 Architecture mecatronique", ...
  - "2 CARTOGRAPHIE REGLEMENTAIRE", "2.1 Matrice de croisement", ...
  - jusqu'a "7 RECOMMANDATIONS STRATEGIQUES AEGIS", "7.1 Niveau de risque global", ..., "7.5 Sources reglementaires citees"
  - SANS prefixe parasite, SANS doublon de numerotation, SANS "SECTION N :" dans le texte du titre.

Si tu te sens tente d'ecrire `## SECTION 1 : ...` ou `### 1.1 Description...`, ARRETE-TOI : c'est INTERDIT. La numerotation est automatique LaTeX.
